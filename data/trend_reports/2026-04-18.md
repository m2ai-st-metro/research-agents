# Weekly Trend Report: April 5, 2026

## Executive Summary
Developer tools and agent skills are gaining traction, with MCP and sandboxing emerging as hot topics. However, discussions around MCP-servers and agent-skills are cooling down, suggesting a shift in focus towards more specialized and secure tools.

- **MCP**: Increased interest in MCP with a 37% rise in signals, driven by updates and new features like the 2026 roadmap (e.g., GitHub MCP server updates).

## Falling Themes
